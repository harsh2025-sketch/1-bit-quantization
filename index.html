Abstract

Recent advancements in extreme low-bit serialization have demonstrated the viability of 1.58-bit ternary large language models for minimizing computational and memory footprints. However, standard low-bit architectures introduce a hypothesized degradation mechanism we term Boundary Representation Collapse at the embedding interfaces, suffer from signal amplitude decay at extreme depths, and fail to address the memory asymmetry wherein continuous 16-bit Key-Value (KV) cache tensors outgrow weight parameters at extended context scales.

We present three co-dependent systems proposals: Hourglass Quantization, Sub-Norm Gain Compensation, and Ternary-Aligned Chain-of-Thought (TaCoT). Preliminary empirical validation over a 6,000-step training regime utilizing a modified Qwen2.5-0.5B architecture provides initial feasibility evidence.

Introduction

The scaling laws of deep autoregressive language models have traditionally bound raw performance to parameter density. Extreme low-bit serialization—specifically 1.58-bit ternary quantization—has emerged as a disruptive alternative. By constraining weight matrices to discrete values, ternary networks replace expensive floating-point matrix multiplications with highly efficient integer addition and subtraction operations.

Methodology

To mitigate semantic parsing degradation, we establish an asymmetric configuration termed Hourglass Quantization. The input embedding layer and the terminal language modeling head are explicitly isolated in continuous high-precision floating-point formats, while internal parameter tensors are mapped to the discrete ternary regime.

Our framework introduces a layer-dependent analytical gain modifier \( \alpha_i \) injected directly into the standard RMSNorm step:

\[ \alpha_i = 1.0 + \left(8.0 \cdot \frac{i}{N - 1}\right) \]

To prevent KV cache from overwhelming edge VRAM, we propose a Sink-Anchored Sliding Window Attention (SWA) matrix layout.

Results

The 6,000-step training loop executed without numerical explosion. The cross-entropy loss exhibited an initial decay before rising by step 6000. A direct memory tensor extraction verified the structural implementation of the Hourglass Quantization topology, counting exactly 3 unique values within an internal attention matrix during the active forward pass.
